# Triage Hints: Digital Mobile Radio (DMR, P25, TETRA, NXDN, D-STAR)

These hints help the LLM identify and verify digital land mobile radio signals from statistical reports generated by `triage_iq.py`.

---

## 🔍 Key Spectral Indicators

### 1. Occupied Bandwidth (OBW)
Channel bandwidth is the strongest first-pass discriminator between digital radio families:

| Protocol | Channel BW | 99% OBW |
|---|---|---|
| **DMR** | 12.5 kHz | ~10–11 kHz |
| **P25 Phase 1** | 12.5 kHz | ~10–11 kHz |
| **P25 Phase 2** | 12.5 kHz | ~10–11 kHz |
| **TETRA** | 25 kHz | ~20–22 kHz |
| **NXDN48** | 6.25 kHz | ~5 kHz |
| **D-STAR DV** | 6.25 kHz | ~5 kHz |

* **12.5 kHz signals**: DMR, P25 Phase 1, P25 Phase 2, NXDN96
* **25 kHz signals**: TETRA (the only common digital LMR at 25 kHz)
* **6.25 kHz signals**: NXDN48, D-STAR DV

### 2. PAPR (Peak-to-Average Power Ratio)
* **4FSK (DMR, P25 C4FM, NXDN)**: Constant envelope → PAPR **~1.5–3.0 dB** (very low).
* **π/4 DQPSK (TETRA)**: Near-constant envelope with controlled zero-crossings → PAPR **~3.0–4.5 dB**.
* **H-DQPSK (P25 Phase 2)**: Slightly higher → PAPR **~3.5–5.0 dB**.
* **GMSK (D-STAR)**: Constant envelope → PAPR **~1.0–2.0 dB**.

### 3. Spectral Shape
* **4FSK signals** (DMR, P25 Ph1): Show **4 distinct spectral humps** corresponding to the 4 deviation levels. DMR shows peaks at ±648 Hz and ±1944 Hz from center.
* **TETRA (π/4 DQPSK)**: Smooth, raised-cosine shaped spectrum filling the full 25 kHz channel. No discrete FSK humps.
* **D-STAR (GMSK)**: Very compact, smooth Gaussian-shaped spectrum within ~5 kHz.

### 4. OFDM Flatness Score
* All digital LMR protocols are **single-carrier** modulations.
* Flatness score will be **low (<0.4)** for all protocols in this family.
* If flatness is high (>0.7), it is not a digital LMR signal.

## ⏱ Key Temporal Indicators

### 1. TDMA Slot Timing (Smoking Gun for DMR & TETRA)

| Protocol | Slot Duration | Frame Duration | Slots per Frame |
|---|---|---|---|
| **DMR** | 30 ms | 60 ms | 2 |
| **P25 Phase 2** | 30 ms | 60 ms | 2 |
| **TETRA** | 14.167 ms | 56.67 ms | 4 |
| **P25 Phase 1** | Continuous | — | 1 (FDMA) |
| **D-STAR** | Continuous | — | 1 (FDMA) |

* **DMR/P25 Ph2**: Bursts of exactly **27.5 ms** active + **2.5 ms** guard, repeating at **30 ms** intervals. If two interleaved patterns are visible, this confirms 2-slot TDMA.
* **TETRA**: Four bursts per **56.67 ms** frame. The 4-slot pattern at 25 kHz bandwidth is unique to TETRA.
* **P25 Phase 1 / D-STAR**: Continuous transmission (no TDMA gaps) — bursts will appear as solid blocks.

### 2. Transmission Duration
* Voice calls typically last **3–60 seconds**.
* Control channel transmissions are brief (**50–200 ms**) and periodic.
* Trunking systems show periodic control channel bursts at regular intervals (e.g., every 60 ms for DMR, every 56.67 ms for TETRA).

## 🧬 Autocorrelation Periodicities

### DMR
* Strong peaks at lags corresponding to the **60 ms** TDMA frame:
  - At 2 MSPS: $60\\ \text{ms} \times 2000 = \mathbf{120{,}000\\ \text{samples}}$
* Secondary peaks at **30 ms** (single slot):
  - At 2 MSPS: $\mathbf{60{,}000\\ \text{samples}}$

### TETRA
* Strong peaks at the **56.67 ms** frame period:
  - At 2 MSPS: $56.67\\ \text{ms} \times 2000 = \mathbf{113{,}333\\ \text{samples}}$
* Sub-peaks at **14.167 ms** (single slot):
  - At 2 MSPS: $\mathbf{28{,}333\\ \text{samples}}$

### TETRA Multiframe
* Strong autocorrelation at the **1.02 s** multiframe boundary (18 frames):
  - At 2 MSPS: $\mathbf{2{,}040{,}000\\ \text{samples}}$

## ⚠️ Common False Positives

| Confounder | How to Distinguish |
|---|---|
| **Analog FM voice** | FM voice has continuous, variable-amplitude spectrum without TDMA gaps. No sync word correlation. PAPR similar but no discrete symbol levels in discriminator output. |
| **POCSAG paging** | POCSAG uses 2-FSK (2 levels) not 4-FSK. Has very long alternating-bit preamble (>200 ms). Baud rate is 512/1200/2400, not 4800. |
| **dPMR** | dPMR uses 4FSK at 4800 sym/s like DMR but uses FDMA only (no TDMA) and 6.25 kHz channels. No TDMA gaps visible. |
| **DMR vs P25 Phase 1** | Both are 4FSK at 4800 sym/s in 12.5 kHz. Differentiate by sync pattern (DMR: 48-bit patterns; P25: `0x5575F5FF77FF`). P25 C4FM deviation is ±1800/±600 Hz vs DMR ±1944/±648 Hz. |
| **TETRA vs wide FM** | TETRA has rigid 56.67 ms TDMA framing. FM voice has no periodic temporal structure. |

## 🎯 Quick Decision Tree

```
1. Channel bandwidth?
   ├─ ~6.25 kHz → NXDN48 or D-STAR
   │   └─ GMSK (smooth spectrum) → D-STAR
   │   └─ 4FSK (4 humps) → NXDN
   ├─ ~12.5 kHz → DMR, P25, or NXDN96
   │   └─ TDMA gaps at 30 ms intervals? → DMR or P25 Phase 2
   │   │   └─ Sync = 0x755FD7DF75F7? → DMR
   │   │   └─ Sync = 0x5575F5FF77FF? → P25
   │   └─ Continuous (no TDMA) → P25 Phase 1 or NXDN96
   │       └─ 4800 sym/s, ±1800/±600 Hz dev → P25 Phase 1
   │       └─ 2400 sym/s, ±1050/±350 Hz dev → NXDN96
   └─ ~25 kHz → TETRA
       └─ 4-slot TDMA at 56.67 ms frame? → Confirmed TETRA
```

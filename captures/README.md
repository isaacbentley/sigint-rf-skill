# DefCon Challenge Captures

Drop a capture file here (for example, `mystery_capture.cf32`) and ask your agent to take a look — *"what is this signal?"*

**No radio yet?** Just ask your agent for a practice signal — *"make me a demo capture to play with"* — and it'll drop one right here for you to triage and decode.

See the Quick Start section of the [README](../README.md) to get started.
